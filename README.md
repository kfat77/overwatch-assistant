# 守望先锋辅助插件 Overwatch Assistant

一个为《守望先锋》设计的游戏辅助工具，**不注入游戏进程，使用屏幕截图+OCR+透明叠加层**，完全安全无封号风险。

## 功能

### 1. 实时聊天翻译
- 自动识别聊天框中的 **英文** 和 **韩文** 消息
- 实时翻译为 **中文** 显示在叠加层上
- 支持游戏术语快速替换（如 gg -> 打得好, heal -> 治疗我）

### 2. 英雄推荐系统
- 根据队友已选英雄推荐最适合的阵容搭配
- 分析阵容完整性（重装/输出/支援数量）
- 基于英雄协同关系给出推荐（如 法老之鹰+天使）
- 支持克制关系分析

## 技术方案

```
┌─────────────────────────────────────────────────────────┐
│                      守望先锋游戏画面                       │
│  ┌─────────────────┐                                    │
│  │   聊天框区域     │  ←── 截图 ──→ OCR识别 ──→ 翻译      │
│  └─────────────────┘                                    │
│  ┌──────────────────────────────┐                      │
│  │        英雄选择界面            │  ←── 模板匹配/手动输入   │
│  └──────────────────────────────┘                      │
└─────────────────────────────────────────────────────────┘
                           │
                    ┌──────▼──────┐
                    │  透明叠加层   │  ←── 显示翻译和推荐
                    └─────────────┘
```

### 为什么安全？
- **零注入**：不读取游戏内存，不使用 DLL 注入
- **纯截图**：仅使用 Windows API 截取屏幕画面
- **透明窗口**：使用操作系统标准窗口 API 显示叠加层
- **不修改游戏**：不修改任何游戏文件或数据

## 安装

### 前置要求
- Python 3.8+
- Windows 10/11

### 安装依赖

```bash
# 克隆或下载项目
cd overwatch-assistant

# 创建虚拟环境（推荐）
python -m venv venv
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 安装 Tesseract-OCR（可选，用于 tesseract 引擎）
# 下载地址: https://github.com/UB-Mannheim/tesseract/wiki
# 安装后添加到系统 PATH
```

### OCR 引擎选择

#### 推荐：EasyOCR（韩文识别效果好）
```bash
pip install easyocr
```
第一次运行时会自动下载模型文件。

#### 备选：Tesseract
```bash
pip install pytesseract
```
需要额外安装 Tesseract-OCR 本体。

## 使用指南

### 启动程序

```bash
python main.py
```

### 首次使用步骤

1. **启动守望先锋**，进入训练靶场或等待大厅
2. **按 F11**，拖拽选择聊天框区域
3. **按 F10**，开始聊天翻译
4. **按 F9**，显示/隐藏翻译叠加层

### 英雄推荐使用

1. 进入英雄选择界面
2. **按 F12** 选择英雄选择区域（或手动输入阵容）
3. 程序会自动分析队友阵容并给出推荐

### 热键列表

| 热键 | 功能 |
|------|------|
| `F9` | 显示/隐藏叠加层 |
| `F10` | 开始/停止聊天翻译 |
| `F11` | 选择聊天框区域 |
| `F12` | 选择英雄选择区域 |
| `Ctrl+Shift+Q` | 退出程序 |

## 配置

编辑 `config.py` 调整以下设置：

```python
# 翻译设置
target_language = "zh"      # 目标语言
translation_engine = "google"  # 翻译引擎

# 叠加层设置
overlay_opacity = 0.85      # 透明度
overlay_position = (50, 50) # 窗口位置
message_ttl = 15.0          # 消息停留时间（秒）

# OCR设置
ocr_engine = "easyocr"      # OCR引擎
ocr_languages = ['en', 'ko'] # 识别语言
```

## 英雄模板

为了实现自动识别英雄选择界面的英雄，需要准备英雄头像模板：

1. 在守望先锋中进入英雄选择界面
2. 使用截图工具截取每个英雄头像（约 100x100 像素）
3. 保存到 `assets/heroes/` 目录，文件名为英雄英文名：
   - `Genji.png`
   - `Mercy.png`
   - `Winston.png`
   - ...等等

> 如果没有模板，可以通过**手动输入**方式告诉程序队友选了什么英雄。

## 常见问题

### Q: 为什么识别不出文字？
A: 请确保：
1. 正确选择了聊天框区域（F11）
2. 聊天框文字清晰可见
3. OCR 引擎正确安装

### Q: 翻译速度很慢？
A: 
- 降低截图频率：修改 `capture_interval = 0.5`
- 使用更快的翻译引擎
- 确保网络连接正常

### Q: 叠加层被游戏覆盖？
A: 
- 尝试以管理员身份运行程序
- 在游戏设置中使用"窗口化全屏"模式
- 右键点击叠加层选择"置顶"

### Q: 如何添加更多语言？
A: 修改 `config.py` 中的 `ocr_languages`：
```python
# 日文
ocr_languages = ['en', 'ko', 'ja']
```

## 注意事项

1. **本工具仅读取屏幕像素，不注入游戏进程**
2. **建议在非排位模式中先测试功能**
3. **叠加层窗口可能被某些全屏游戏覆盖**，建议使用窗口化全屏模式
4. **OCR 识别率受分辨率、字体、背景影响**，可能需要调整截图区域
