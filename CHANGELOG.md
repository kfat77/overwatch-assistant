# Changelog

所有项目的重要更改都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [SemVer](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-XX

### 初始版本

#### 功能
- 实时聊天翻译（英文、韩文 → 中文）
- 游戏术语快速替换（gg→打得好, heal→治疗等）
- 英雄阵容推荐系统
  - 基于英雄协同关系推荐
  - 基于敌方克制关系推荐
  - 阵容完整性分析
- 透明叠加层显示
- 屏幕截图 + OCR 文字识别
  - 支持 EasyOCR 和 Tesseract
  - 图像预处理增强
- 可视化区域选择工具

#### 技术特性
- 零游戏注入，纯截图方案
- 支持多分辨率缩放
- 可配置的翻译引擎和 OCR 引擎
- 翻译结果缓存
- 文本去重机制

#### 支持语言
- 源语言：英文、韩文
- 目标语言：中文

#### 支持的英雄数据
- 全部 40+ 守望先锋英雄
- 完整的中文名映射
- 英雄协同关系数据库
- 英雄克制关系数据库
- 常见阵容策略（放狗/地推/消耗/冲脸）
